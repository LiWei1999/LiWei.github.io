机器人运动学复习

发表于2021-07-21|更新于2021-07-23
|阅读量:

​   这是大三上学期《机器人技术基础》课程当中有关于机器人运动学部分的练习实验，主要涉及到D-H模型建立，运动学正逆解和经典的取放轨迹规划方法（在这里用的是5-3-5轨迹）。

D-H模型搭建

​   机器人一般是由一系列的关节和连杆按任意顺序连接而成的。我们对每个关节指定一个参考坐标系，然后确定从一个关节到下一个关节进行变换的步骤，这样我们就可以得到机器人总变换矩阵。

​   D-H表示法对于每个关节，都必须指定一个z轴和x轴，通常并不需要指定y轴，因为y轴总是垂直于x轴和z轴的，此外，D-H表示法也根本不需要用到y轴。其表示法指定本地参考坐标系步骤如下：

  1. 对于所有的关节，全都用z轴来表示。如果关节是旋转的，那么z轴位于右手规则旋转的方向，绕z轴的旋转角$\theta$为关节变量；而如果关节是滑动的，则z轴为沿直线运动方向，沿z轴的连杆长度d为关节变量。关节n处的z轴编号为n-1，即表示关节n+1运动的z轴为$z_n$

  2. 确定好z轴之后确定x轴。通常关节不一定平行或相交，但是我们总能找到一条距离最短的公垂线正交于任意两条斜线。所以在公垂线方向上定义本地参考系的x轴，用$a_n$表示$z_{n-1}$与$z_n$之间的公垂线，定义$x_n$方向为沿$a_n$方向。当然两个相邻坐标系之间存在一些特殊的位置关系需要具体讨论：

    1. 如果两个关节的z轴平行，那么它们之间就有无数条公垂线，这时就可以挑选与前一关节的公垂线共线的一条公垂线，便于简化模型。
    2. 如果相邻两个关节的z轴是相交的，那么他们之间就没有公垂线，此时就可将垂直于两条轴线构成的平面的直线指定为x轴。即选取两条z轴的叉积方向作为x轴。

  如下图则是根据上述规则建立出来的模型，$\theta$表示绕z轴的旋转角，d表示在z轴上两条相邻的公垂线之间的距离（或称关节偏移），a表示每一条公垂线的长度（连杆长度），$\alpha$表示相邻两个z轴之间的角度（扭角）。通常，只有$\theta$和d是关节变量。

  那么只要依此进行几个必要的运动，就可将一个参考坐标系变换到下一个参考坐标系。例如现在要从本地参考坐标系$x_n - z_n$变换到下一个本地参考坐标系$x_{n+1} - z_{n+1}$，只需要经过以下四个步骤即可：

  1. 绕$z_n$轴旋转$\theta_{n+1}$,使得$x_n$和$x_{n+1}$相互平行。因为$a_n$和$a_{n+1}$都是垂直于$z_n$轴的，因此绕$z_n$轴旋转$\theta_{n+1}$确实能使$x_n$和$x_{n+1}$相互平行。
  2. 沿$z_n$轴平移$d_{n+1}$距离，使得$x_n$和$x_{n+1}$共线，因为$x_n$和$x_{n+1}$已经平行且垂直于$z_n$轴，则沿着$z_n$轴移动可使他们相互重叠在一起。
  3. 沿已经旋转过的$z_n$轴平移$a_{n+1}$的距离，使得$x_n$和$x_{n+1}$的原点重合。此时两个参考坐标系的原点处于同一位置。
  4. 将$z_n$轴绕$x_{n+1}$轴旋转$\alpha_{n+1}$，使得$z_n$和$z_{n+1}$轴对准，这时坐标系n和n+1完全相同。

运动学正逆解

  正逆解用于关节角空间和直角坐标空间位姿的相互解。运动学正解为在已知机器人各个关节角的情况下，求解出此时机器人末端在空间当中的位姿；而逆解则更加重要一些，给定机器人末端在空间中的位姿信息，反向推导出机器人各个关节角的大小。

取放轨迹规划

问题描述

  掌握典型的取-放轨迹规划方，完成5-3-5实现的典型取-放轨迹，其中取放作业应包含起始位置（取点）、转移位置（抬起或离开点）、卸下位置（下降点）和目标位置（放置点），整理出边界条件和连续条件，列出轨迹未知系数的推导过程。然后，使用Matlab Robotics 取得PUMA560机器人末端的任一组取-放作业的四个位置点，经逆解得各关节的位置。此外，假设起始位置、目标位置的关节速度和加速度都设为0，三段的运动时间分别是2s，4s和3s。接着，使用Matlab求解出轨迹的未知系数，生成PUMA560各关节的位置、速度和加速度的轨迹函数。最后，将这些轨迹输入Matlab Robotics的PUMA560中，仿真执行取放轨迹，观察其与三段直线轨迹的执行结果的差别并分析原因。

确定各点速度

  我们给定以下的取放任务的各点位置：

起始位置：aim0 = [0,-0.5,-0.5];
离开位置：aim1 = [0,-0.5,0.2];
下降位置：aim2 = [-0.5,0.5,0.2];
放置位置：aimx = [-0.5,0.5,-0.5];

  对于每个位置可以进行求逆解，我们就可以得到在每个位置上puma560机器人六个关节相对应的角度。要想得到5-3-5轨迹方程的未知系数，还需要知道两个中间点的速度，我们采用启发式方法确定关节速度，用直线段把这些结点连接起来，若相邻线段的斜率在结点处改变符号，就设其速度为0；若斜率不改变符号，则其速度设为结点两侧线段斜率的平均值。通过上面得到的位置信息，我们可以很轻松地算出两个中间点的速度矩阵。加速度在关键点处设为0，简化计算。
